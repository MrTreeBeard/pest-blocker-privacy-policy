Privacy Policy — Pest Blocker

Last updated: 2026-01-12
Summary: Pest Blocker does not collect, store, transmit, or sell personal data.

What data Pest Blocker collects

Pest Blocker does not collect personal data.

What data Pest Blocker stores locally

The extension stores user-controlled settings locally using Chrome extension storage, such as:

  • Blocked domain list
  • Popup allowlists (global and per-site)
  • Popup policy mode and temporary disable state
  • Optional local block log (if enabled)

This data stays on the user’s device (or in the user’s Chrome sync storage if enabled by the user) and is used only to provide the extension’s functionality.

What data Pest Blocker transmits

Pest Blocker does not transmit data to external servers. It does not use analytics, tracking, or advertising SDKs.

Permissions

Pest Blocker requests permissions such as declarativeNetRequest, host access, tabs, and storage solely to:

  • Block user-specified domains at the network layer
  • Prevent and close unwanted popup/new-tab behavior
  • Allow the user to manage block/allow lists and settings

Remote code

Pest Blocker does not use remote code. All code is packaged with the extension.

Data sharing and sale

Pest Blocker does not sell, rent, or share user data with third parties.

Changes to this policy
